## üß≠ Modes to Know

* **Edit Mode** ‚Üí You're typing inside a cell (`green bar`)
* **Command Mode** ‚Üí You're controlling cells (`blue bar`)

Toggle between them:

| Action             | Shortcut |
| ------------------ | -------- |
| Enter Edit Mode    | `Enter`  |
| Enter Command Mode | `Esc`    |
